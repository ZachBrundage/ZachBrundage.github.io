Important Information:

This app pulls punchlist data from the Basecamp 3 API and reformats it so that it can be printed for distribution. The Basecamp 3 API DOES NOT currently support CORS, so this app will not work unless you disable CORS restrictions in your browser. See links below for instructions on how to disable CORS in your browser. Please be aware that CORS is enabled in most browsers by default for your safety, so be sure to re-enable CORS on your browser once you are finished using the app.
